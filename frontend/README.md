# DealFlow тАФ Frontend

Website for DealFlow, a Nairobi-based digital services agency offering web design, KRA/government services, IT support, POS systems, and more.

## Run locally

```bash
npm install
cp .env.example .env
npm run dev
```

Set `VITE_BACKEND_URL` in `.env` to point to the backend (default: `http://localhost:4000`).

## Pages

- `/` тАФ Home
- `/services` тАФ All services
- `/packages` тАФ Website packages & pricing
- `/about` тАФ About
- `/contact` тАФ Contact / get a quote
- `/privacy`, `/terms`, `/refund` тАФ Legal
